# /kb-add — Ajouter une note au vault

Ajouter une nouvelle note à la Scalezia Knowledge Base.

## Instructions

Le user fournit le contenu à ajouter. Détermine automatiquement :

1. **Le type** : framework, idée, méthode, cas-client, source, post, épisode
2. **La destination** :
   - Framework original → `01-Fondations/Frameworks/`
   - Principe / conviction → `01-Fondations/Principes/`
   - Intellectuel (philo, systèmes) → `01-Fondations/Intellectuel/`
   - Méthodologie opérationnelle → `02-Systèmes/{domaine}/`
   - Idée atomique → `03-Intelligence/Idées/Inbox/` (puis promotion vers Permanentes quand validée)
   - Apprentissage / insight → `03-Intelligence/Apprentissages/Par-Theme/`
   - Cas client → `03-Intelligence/Cas-Clients/`
   - Post LinkedIn → `04-Production/LinkedIn/`
   - Source externe → `05-Sources/`
3. **Les tags** adaptés : `type/`, `domain/`, `phase/`, `pilier/`, `status/`
4. **Les wikilinks** vers les notes existantes dans le vault

Utilise le template approprié depuis `_Templates/`.

Écris le fichier, confirme la destination et les liens créés.
